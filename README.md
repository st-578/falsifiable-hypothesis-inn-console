# falsifiable-hypothesis-inn-console

VSCode Extension for **falsifiable-hypothesis-inn**

## コマンド
- `Falsifiable: 新規仮説生成` → 新しいMarkdownファイルに反証可能仮説テンプレートを作成
- `Falsifiable: この仮説をフォーク提案` → メインリポジトリを開く
- `Falsifiable: X投稿テンプレート生成` → X用投稿文をクリップボードにコピー

## インストール方法（開発モード）
1. このリポジトリをクローン
2. VSCodeで開く
3. `F5` で Extension Development Host を起動
4. 新しいウィンドウでコマンドパレット（Ctrl+Shift+P）から「Falsifiable:」で検索

本番公開は後でVSCode Marketplaceに載せます。
